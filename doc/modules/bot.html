cfg (options)
Initializes the bot with options

Parameters:

  • options (table) Options
    • token (string) Bot token (optional)
    • parse_mode (string) Parse mode. HTML by default (optional)
    • api_url (string) (optional)

Returns:

    bot object

Usage:

    bot:cfg {
 token = '1234567:AABBccDDFF',
 parse_mode = 'HTML'
}
call (method, options[, request_param])
Executes a Telegram Bot API method.

Parameters:

  • method (string) TG API method to execute
  • options (table) Method options
  • request_param (table) { multipart_post = true } (optional)

Returns:

  1. (table) Response from the Telegram Bot API
  2. (table) Error object

Usage:

    bot:call("sendMessage", {
 text = 'Hello!',
 chat_id = 123456789,
})

send_certificate (options)
Sends a certificate for webhook setup

Parameters:

  • options (table) Options table
    • url (string) URL for the webhook

startWebHook (options)
Start the webhook

Parameters:

  • options (table) Options table
    • host (string) Host to bind to (default is '0.0.0.0')
    • port (number) Port to listen on (default is 9091)
    • path (string) Route path ('/' string by default)
startLongPolling ([options])
Start long polling

Parameters:

    • allowed_updates (table) (optional)
