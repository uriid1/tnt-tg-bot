init (bot, max_connections)
Initialize the request module with a bot instance and maximum connections.

Parameters:

  • bot The bot instance.
  • max_connections The maximum number of connections (optional, default is 100).

Returns:

    The initialized request module.

Send an HTTP request to the Telegram Bot API.

Parameters:

Returns:
