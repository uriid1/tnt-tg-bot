# Mini Shop

Минимальный шаблон реализации магазина.

Пример не доделан, но хорошо показывает реализацию пагинации и того как можно выстроить структуру проекта.

## Запуск
  1. Выполнить `bash tnt-tg-bot.pre-build.sh` для установки зависимостей (.rocks директория вынесена в симлинк)
  2. Перейти в директорию `tnt-tg-bot/examples/mini-shop`
  3. Выполнить запуск с переменной окружения
```bash
BOT_TOKEN="BOT TOKEN HERE" ./scripts/start
```

## Заполнение БД пустышками
> [!NOTE]
> Для начала выполнить запуск, после чего остановить и выполнить эту команду.

```bash
tarantool fill_space.lua
```
