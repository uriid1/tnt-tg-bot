# Паттерн Observer (Наблюдатель)

Поведенческий шаблон проектирования, предназначенный для организации односторонней зависимости между объектами: когда один объект (Subject) изменяет своё состояние, все зависимые от него объекты (Observers) автоматически получают уведомление и реагируют на это.

## Запуск
```bash
BOT_TOKEN="BOT TOKEN HERE" tarantool init.lua
```
